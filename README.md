## Description
This is a simple console-based Rock-Paper-Scissors game written in C++.

- Writing modular functions
- Handling basic game logic and user input

- Player vs Computer gameplay
- Tracks number of wins, losses, and draws
- Multiple rounds support
- Colorful console output for game result feedback
